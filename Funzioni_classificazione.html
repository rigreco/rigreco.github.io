Classificazione delle funzioni

Funzioni matematiche
Funzioni algebriche
Funzioni razionali intere (Polinomi) Forma: $ax^n + bx^{n-1} + \ldots + c$
Dominio: $\mathbb{R}$
Funzioni razionali fratte ($\frac{A(x)}{B(x)}$) Dominio: $\mathbb{R} \setminus \{x \mid B(x) = 0\}$
Funzioni irrazionali ($\sqrt[n]{f(x)}$) Dominio:
  • $n$ dispari: $\mathbb{R}$
  • $n$ pari: $\{x \mid f(x) \geq 0\}$
Funzioni trascendenti
Funzioni esponenziali ($a^x$) Dominio: $\mathbb{R}$
Funzioni logaritmiche ($\log_a(x)$) Dominio: $\{x \mid x > 0\}$
Funzioni goniometriche Seno ($\sin(x)$): Dominio: $\mathbb{R}$
Coseno ($\cos(x)$): Dominio: $\mathbb{R}$
Tangente ($\tan(x)$): Dominio: $\mathbb{R} \setminus \{\frac{\pi}{2} + k\pi \mid k \in \mathbb{Z}\}$
